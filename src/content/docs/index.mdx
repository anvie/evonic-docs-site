# Evonic AI

Local-first agentic AI for open models.

## Features

- **Local-first** — run everything on your machine
- **Agent framework** — create and manage autonomous agents
- **Plugin system** — extend with custom tools, channels, and events
- **Skill system** — modular capabilities that agents can use
- **Scheduler** — schedule tasks and runs
- **Evaluation framework** — test and benchmark agent performance
- **Model support** — Ollama, vLLM, and local models

## Security

Evonic AI prioritizes security with:

- Isolated agent workspaces
- Heuristic safety on tool execution
- Strict context management

## Quick Start

Install Evonic AI and create your first agent:

```bash
pip install evonic
evonic init my-agent
evonic run my-agent
```
