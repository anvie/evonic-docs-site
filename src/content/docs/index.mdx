## Three Differentiating Concepts

Evonic stands apart from other agentic frameworks through three foundational design decisions:

1. **Workplace** — Agents execute in isolated environments that can be local directories, remote SSH servers, or cloud devices connected via Evonet. No public IP or complex firewall configuration needed.

2. **Agent-to-Agent Communication** — Agents communicate natively as a first-class protocol. Swarm orchestration, role-based delegation, and multi-step collaboration are built into the runtime, not bolted on.

3. **Heuristic Mal-activity Detection System** — Every agent action is inspected through a multi-layer safety system before execution. Dangerous patterns are caught and blocked, and suspicious behavior escalates to human approval — no silent failures.

## Quick Start

Get up and running with a single command:

```bash
curl -fsSL https://evonic.dev/install.sh | bash
```

Then:

1. **Configure**: Set your API key and model
2. **Create an agent**: Define a system prompt and tools
3. **Run**: Start the server and test
