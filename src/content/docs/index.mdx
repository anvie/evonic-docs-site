## Features

<CardGrid stagger>
  <Card title="Open Model Optimized Agentic Engine" icon="rocket">
    Agent runtime optimized for open/local models (Ollama, llama.cpp, vLLM). Session management, LLM orchestration, and event-driven architecture.
  </Card>
  <Card title="Secure by Default" icon="shield-check">
    Isolated agent workspaces, heuristic safety on tool execution, and strict context management. Plan/Execute mode for full control.
  </Card>
  <Card title="Evaluator Battery Included" icon="list-format">
    Multi-domain evaluation (conversation, math, SQL, tool calling, reasoning, health) with 5 complexity levels. Explore open-source models to find the right fit.
  </Card>
  <Card title="Multi-Channel" icon="comment-alt">
    Connect agents to Telegram, WhatsApp, Discord, and more through a modular channel abstraction.
  </Card>
  <Card title="Skills" icon="puzzle">
    Pluggable capability system. Skill packages are self-contained (skill.json + setup.py + backend/tools/). Upload via web UI or CLI.
  </Card>
</CardGrid>
